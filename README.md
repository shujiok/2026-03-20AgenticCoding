# 2026-03-20AgenticCoding

小学生向けの「ハノイの塔」学習Webアプリです。

## できること

- 円盤の枚数を変えて挑戦できる
- 棒をクリックして自分で動かせる
- ルール違反をやさしく案内してくれる
- 最短手数と現在手数を見比べられる
- 自動で解くアニメーションを見られる

## 使い方

1. `index.html` をブラウザで開く
2. 円盤の枚数を選ぶ
3. 棒を2回クリックして円盤を移動する
4. 「お手本を再生」で自動解法を見る

## ファイル構成

- `index.html` : 画面構成
- `styles.css` : 見た目とアニメーション
- `script.js` : ゲームロジックと自動解法
